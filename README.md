# Beer API

> RESTful API with Unit testing for Service and Controller layers

## :wrench: &nbsp;&nbsp; Technologies

- Java 11
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- JUnit
- Mockito
- Swagger

## :bulb: &nbsp;&nbsp; API

This API can perform CRUD operations on an entity (beers) and perform unit testing on all those operations.
